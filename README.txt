🎫 好きなことログ & To Do リスト（デモ版）

概要
このデモ版テンプレートは、観劇・ライブ・ミュージカルなどの「好きなこと」を記録するWebアプリの簡易体験版です。

本番テンプレートに含まれるさまざまな機能の一部を、保存なしで気軽に試すことができます。


✅ このデモで体験できること
🎨 テンプレート（標準／舞台／ライブ）の切り替え
🧳 観劇前のTo Doリストの追加・削除
📝 観劇メモの入力
🌙 ダークモードのON／OFF


🚫 このデモでは利用できない機能（製品版にて対応）
💾 入力データの保存（タイトル／メモ／ToDoなど）
📷 写真アップロードと保存機能
🔐 バックアップの保存・復元
✅ お気に入りポイント（チェックリスト）
🎯 「次に観たい作品」記録欄
🔄 タイトルごとの情報切り替え


特長（製品版と共通）
スマホ対応のレスポンシブデザイン
HTML／CSS／JavaScriptで構成されたシンプル設計
GitHub Pagesなどでの公開にも対応


利用方法（デモ版）
index.html をブラウザで開くだけで動作します。
テンプレートを切り替えて見た目を確認。
To Do リストやメモ欄に入力して使い心地を体験。

💡 入力内容は保存されません。体験後はリロードでリセットされます。
